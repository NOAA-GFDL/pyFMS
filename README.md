# **`fms2py`**

## **Installation**
`pyFMS` requires a compiled cFMS library.  The library can be installed
by 

1.  git clone --recursive https://github.com/NOAA-GFDL/pyFMS.git
2.  cd pyFMS
3.  python ./compile.py

The script `compile.py` will first compile FMS within the cFMS submodule
directory
